validate_patient_data module¶

validate_patient_data.validate_patient_data(data_raw)¶

This function checks for invalid data when posting a new patient info.

The function takes user input and check for errors. It can accommodate some mistakes such as typing the “id” instead of patient_id, entering a string instead of a number, etc.

Parameters:data_raw (dict) – A dictionary containing user information.
Returns:A dictionary containing validated user information.
Return type:dict
